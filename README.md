# 🐚 Shell++
A simple shell written in C++.

## 🔨 Building
### 📝 Dependencies
- [GCC](https://gcc.gnu.org/)
- [Make](https://www.gnu.org/software/make/)

To build the shell, run the following commands:
```bash
make
```

## 📖 Usage
To run the shell, run the following command:
```bash
./shell
```
